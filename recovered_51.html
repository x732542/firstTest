Module 01 · HTML
\n
HTML 常用標籤教學工具
\n
\n
\n
\n

\n 以分類索引整理最常用的 HTML 標籤，每個標籤包含語意說明、類比、解剖、樣式範例與 JS 連結，讓使用者直接修改範例程式碼並即時預覽結果。\n

\n
標籤分類
\n
\n
\n
文件基礎
\n
html, head, body, title, meta, link, script
\n
\n
\n
版面語意
\n
header, nav, main, section, article, aside, footer, div
\n
\n
\n
文字排版
\n
h1~h6, p, span, a, strong, em, blockquote
\n
\n
\n
表單互動
\n
form, input, button, select, textarea, label
\n
\n
\n
\n
\n
每個標籤包含的內容
\n
\n
\n 語意\n 標籤的正確使用場景與語意說明\n
\n
\n 類比\n 用生活比喻幫助理解抽象概念\n
\n
\n 解剖\n 語法組成與各部分作用說明\n
\n
\n 即時預覽\n 直接修改範例程式並觀察渲染結果\n

Module 03 · JavaScript
\n
JS 邏輯實驗室
\n
\n
\n
\n

\n 延續 HTML 教學頁的版面規格，保留 JavaScript 頁面的暖橘互動感與編輯器氛圍。左側挑主題，中間看觀念，右側直接執行範例與觀察狀態變化。\n

\n
實驗室功能
\n
\n
\n \n 可直接編輯的程式碼區\n Ctrl+Enter 快捷鍵執行\n
\n
\n \n 互動預覽區\n 按鈕、輸入框、列表、進度條等元件\n
\n
\n \n 內建 Console\n log / warn / error 沙盒輸出\n
\n
\n \n 環境重置\n 一鍵還原預覽區所有狀態\n
\n
\n
\n
\n
沙盒執行機制
\n
\nfunction runJS() {\n // 用 new Function 建立隔離沙盒\n const execute = new Function(\n 'console', 'helpers', code\n );\n execute(sandboxConsole, createHelpers());\n\n // 自動清理 timeout / 事件\n clearRuntime();\n}
\n
\n 每次執行前自動清除前一次的 setTimeout、setInterval 與 addEventListener，確保範例不互相干擾。\n
\n

JS Curriculum · Level 1
\n
課程地圖 · 初階主題
\n
\n
\n
\n 初階\n DOM · 改寫內容與樣式\n querySelector 找節點 → 修改文字 / class / style，立即改變畫面\n
\n
\n 初階\n Event · 點擊事件與狀態切換\n addEventListener 回應使用者操作，是互動式網頁的重要基礎\n
\n
\n 初中階\n Form · 讀取輸入框資料\n input.value.trim() 讀取並同步到其他區塊\n
\n
\n 初中階\n Condition · 條件判斷與分支\n if / else if 依照輸入做出不同反應，is 流程控制最基本的功能\n
\n
\n 中階\n Event · 鍵盤事件互動\n keydown 監聽，讀取 event.key / event.code，做快捷鍵或即時回饋\n
\n
\n
\n\n \n
\n
JS Curriculum · Level 2
\n
課程地圖 · 中階主題
\n
\n
\n
\n 中階\n Array · 陣列渲染清單\n map + join 把資料轉成畫面，前端渲染的常見模式\n
\n
\n 中階\n Object · 物件資料驅動畫面\n 把相關資料綁在一起，單一來源更新多個 DOM 節點\n
\n
\n 中階\n Syntax · 解構賦值快速取值\n { title, category } = obj 快速取出欄位，語法更精簡\n
\n
\n 中階\n Async · 計時器與節奏控制\n setTimeout / setInterval 安排延遲任務，生命週期管理\n
\n
\n 中階\n Form · 前端表單驗證\n trim + 長度條件判斷，立即給使用者格式回饋\n
\n
\n 中高階\n Array · filter 與 reduce 統計\n 先篩選合格資料，再用 reduce 累積計算總和與平均\n
\n
\n
\n\n \n
\n
JS Curriculum · UI Components
\n
課程地圖 · UI 元件主題
\n
\n
\n
\n \n 按鈕切換\n classList 切換 class 狀態，布林值驅動雙按鈕\n
\n
\n \n Tabs 分頁\n 切換 active 狀態並同步更新對應內容\n
\n
\n \n 進度條\n style.width 遞增更新，視覺化完成度\n
\n
\n \n Toast 通知\n 短暫顯示後延遲自動隱藏的反饋元件\n
\n
\n \n Accordion\n 一次展開一格，單開邏輯摺疊面板\n
\n
\n \n Dropdown\n 選單切換 active，同步更新標題與說明\n
\n
\n \n Modal\n 顯示/隱藏對話視窗，內容注入控制\n
\n
\n \n 動畫過場\n transition + transform 製作節奏感視覺回饋\n
\n

JS Curriculum · Level 3–4
\n
課程地圖 · 中高階 & 進階
\n
\n
\n
\n 中高階\n DOM · dataset 自訂資料屬性\n data-* 屬性讓元素攜帶自訂狀態，程式讀取寫入不影響外觀\n
\n
\n 中高階\n Browser · localStorage 本機儲存\n setItem / getItem 儲存資料到瀏覽器，重新整理後仍保留\n
\n
\n 中高階\n Data · 搜尋過濾清單\n 輸入關鍵字 → filter 篩選 → 重新 renderList，即時查找\n
\n
\n 中高階\n Project · Todo Mini 清單互動\n 資料陣列驅動渲染，點擊切換完成狀態並重新渲染\n
\n
\n 進階\n Promise · 非同步結果回傳\n new Promise → resolve → .then() 串接，包裝非同步流程\n
\n
\n 進階\n State · 多步驟狀態切換\n 有限狀態機概念，狀態陣列 + 索引遞進，資料驅動畫面\n
